# Hand Gesture Recognition

## Project Overview
This project develops a Hand Gesture Recognition system using Machine Learning. The model is trained on the LeapGestRecog dataset and can accurately classify different hand gestures from images.

## Dataset
- Dataset: LeapGestRecog
- Source: Kaggle
- Total Images: 20,000+

## Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-Learn
- Matplotlib

## Project Workflow
1. Download and extract the dataset.
2. Load gesture images.
3. Convert images to grayscale.
4. Resize images to 64×64 pixels.
5. Normalize image data.
6. Split data into training and testing sets.
7. Train an SVM classifier.
8. Evaluate model performance.
9. Generate a confusion matrix.
10. Predict gestures from new images.

## Gestures Classified
- Palm
- L
- Fist
- Fist Moved
- Thumb
- Index
- OK
- Palm Moved
- C
- Down

## Model Performance
- Accuracy: 99.9%

## Results
The model successfully recognizes and classifies hand gestures from unseen images with high accuracy.
